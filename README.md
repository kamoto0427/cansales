# Can!セールスとは？
営業する会社とサービスを求める会社をマッチングするアプリです。
アプリの特徴としては、リクエスト型「逆営業」アプリとなっています。
サービスを求める会社(リクエストする会社)が営業する会社にリクエストし、サービスのマッチングができます。

※リクエストとは、「営業してほしい！」と依頼する機能のこと。

アプリの詳細は、
Can!セールスの概要書または、プレゼン資料より参照。
